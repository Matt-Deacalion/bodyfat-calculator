Bodyfat Calculator
==================

A VueJS, web based tool to calculate your bodyfat percentage.
Uses the US Navy method.

**See demo**: `click here`_

.. image:: screenshot.png
    :alt: Bodyfat Calculator
    :align: center


Setup
-----

.. code-block:: bash

    $ npm install


Development
-----------

.. code-block:: bash

    $ npm run serve
